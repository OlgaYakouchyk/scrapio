# Scrapio
##### Назначение системы:
* Предоставление услуг по легальному парсингу запрашиваемых сайтов и сервисов заказчику (Европа, США). 

* Создание легальных платных баз данных (БД) предприятий по разным странам (Европа, США), регионам и отраслям для использования в маркетинговых и иных целях подписчиками.
